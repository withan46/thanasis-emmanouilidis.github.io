Flagship project · ~7 months

Cadastral-Data Decision & Response Pipeline

An end-to-end system that ingests cadastral (land-registry) records, decides the correct outcome for each case, and writes a complete, company-formatted response ready to send — cutting manual handling on a high-volume, Greek-language workflow.

  • Trained a multi-label classification model to categorize incoming cadastral records and route each case toward the correct decision path.
  • Fine-tuned Krikri (Greek LLM), served via Ollama, to generate the final answers in the exact format the business requires.
  • Combined the classifier's structured decision with the fine-tuned model's generation, so the pipeline both chooses the right decision and writes the full response.
  • Deployed and ran the system in-house with vector databases, Docker, and MCP tooling for retrieval, packaging, and orchestration.
